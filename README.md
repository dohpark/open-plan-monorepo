# Open Plan

Turborepo monorepo 프로젝트

## 기술 스택

- **Monorepo**: Turborepo
- **Package Manager**: pnpm
- **Language**: TypeScript

## 워크스페이스

- `apps/web`: Vite + React + TypeScript (구성 예정)
- `apps/storybook`: Storybook (구성 예정)
- `packages/ui`: 공유 UI 컴포넌트 라이브러리

## 스크립트

```bash
# 개발 서버 실행
pnpm dev

# 빌드
pnpm build

# 린트
pnpm lint

# 타입 체크
pnpm check-types

# 포맷팅
pnpm format
```
